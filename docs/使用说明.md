# 刷课系统启动说明

## 快速启动

1. **双击运行** `启动服务器.bat` 文件
2. 等待依赖安装（首次运行）
3. 打开浏览器访问: **http://localhost:8000**

## 手动启动

### 1. 安装依赖（如果需要）
```bash
pip install -r requirements-api.txt
```

### 2. 启动服务器
```bash
python api/main.py
```

## 默认登录信息

| 角色 | 用户名 | 密码 |
| ---- | ---- | ---- |
| 管理员 | admin | admin123 |

## 功能说明

### 套餐定价配置
1. 登录管理员账号
2. 进入管理后台左侧菜单"套餐定价"
3. 设置基础价格和折扣率
4. 系统自动生成三个套餐选项

### 用户功能
- 输入学号密码登录平台
- 选择要刷的课程
- 选择合适的套餐
- 下单并支付

### 智能判断
- 如果课程已完成但考试未完成 → 自动禁止基础版，推荐标准版/尊享版
- 如果仅视频未完成 → 推荐基础版
- 如果两者均未完成 → 推荐尊享版（折扣最优）
